Getting Started
---
### Install dependencies :
```
>  npm install
````

### Start :
```
>  npm start
````

### Build in *Heroku*:
* push
````
>  heroku container:push web -a cencosud-test
````
* release
````
>  heroku container:release web -a cencosud-test
````
